## GlazeWM

A tiling window manager for Windows. Comes with integration with [zebar](https://github.com/glzr-io/zebar) for a status bar.

Install via scoop:
```
scoop bucket add extras
scoop install extras/glazewm
scoop install extras/zebar
```

The config file is located at: `~/.glzr/glazewm/config.yaml`.
